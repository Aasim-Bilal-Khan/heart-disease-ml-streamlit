This project focuses on heart disease prediction using a real-world dataset collected and curated independently. The dataset was gathered from practical settings, then carefully preprocessed to handle missing values, normalize features, and improve overall data quality.

Multiple machine learning models were trained and evaluated on the dataset, including various classification algorithms. Each model’s performance was compared using appropriate evaluation metrics to identify the most accurate and reliable approach.

The best-performing model was selected and deployed through an interactive Streamlit web application. The app allows users to input relevant health parameters and receive real-time predictions, making the model accessible and easy to use for non-technical users.

This project demonstrates the complete machine learning pipeline: data collection, preprocessing, model training, evaluation, selection, and deployment in a user-friendly interface.
